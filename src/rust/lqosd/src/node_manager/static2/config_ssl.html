SSL Setup
Optional HTTPS for the LibreQoS WebUI and API docs using a local Caddy reverse proxy.
How This Works
Choose whether staff will open LibreQoS by hostname or by IP address.

If you enter a public hostname, Caddy will ask Let's Encrypt for a certificate that browsers already trust. Use this when staff can reach LibreQoS by name, like libreqos.example.com.

If you leave the hostname blank, Caddy will secure LibreQoS by IP address with its own local certificate authority. This still encrypts traffic, but each operator computer must trust that local root certificate before browsers stop warning.
