RTT Color Thresholds
Control how latency is colored across the WebUI without changing QoO or QoQ scoring behavior.
These values control how RTT (latency) is colored in heatmaps, map markers, and RTT blocks. They do not change QoO/QoQ scoring.
Threshold Values
Set the transition points for green, yellow, and red RTT ranges.
RTT at or below this value will be colored green.
RTT near this value will be colored yellow.
RTT at or above this value will be colored red.
