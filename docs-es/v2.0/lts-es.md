# Nomenclatura LTS heredada (deprecado)

```{warning}
LTS está deprecado y fue reemplazado por LibreQoS Insight.
Para despliegues actuales y nuevas suscripciones, use la documentación de Insight:
[Insight](insight-es.md)
```

## Nota de migración

Si tiene referencias heredadas a LTS en runbooks/documentación interna:

- Reemplácelas por referencias a Insight.
- Use [insight-es.md](insight-es.md) para comportamiento y flujos actuales.
- Use [api-es.md](api-es.md) para detalles de API de nodo (`/api-docs` en su nodo es la fuente de verdad).
- En algunas rutas UI/código todavía aparecen nombres históricos como `lts_trial` o `config_lts`; en versiones actuales se refieren a flujos de Insight.

Las instrucciones heredadas de onboarding de LTS se removieron para evitar confusión en despliegues nuevos.
