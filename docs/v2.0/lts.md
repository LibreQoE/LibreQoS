# Legacy LTS Naming (Deprecated)

```{warning}
LTS is deprecated and has been superseded by LibreQoS Insight.
For current deployments and new subscriptions, use Insight docs:
[Insight](insight.md)
```

## Migration note

If you have legacy references to LTS in older internal docs/runbooks:
- Replace them with Insight references.
- Use [insight.md](insight.md) for current product behavior and workflows.
- Use [api.md](api.md) for Node API details (`/api-docs` on your node is authoritative).
- In some UI/code paths, historical names such as `lts_trial` or `config_lts` may still exist; these refer to Insight signup/config flows.

Legacy LTS onboarding instructions have been removed from this guide to avoid confusion for new deployments.
